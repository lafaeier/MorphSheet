# 01 — 技术栈与项目结构

## 最终技术选型

| 层级 | 技术 | 版本 | 理由 |
|------|------|------|------|
| 桌面壳 | PyWebView | latest | 轻量原生窗口包裹 HTML 前端，无需安装浏览器 |
| 后端框架 | FastAPI | 0.100+ | 异步支持 + 内置 WebSocket + 自动生成 API 文档 |
| 前端 | Vue 3 + HTML/CSS | 3.x (CDN) | 响应式数据绑定，无需构建工具，CDN 引入即用 |
| 数据处理 | Pandas | 2.0+ | 表格数据的事实标准 |
| Excel 读写 | openpyxl + xlrd/xlwt | latest | openpyxl 负责 xlsx，xlrd/xlwt 负责老旧 xls |
| 编码检测 | chardet | 5.0+ | 自动识别 CSV 编码，处理中文乱码 |
| LLM 客户端 | openai (Python SDK) | 1.0+ | DeepSeek API 兼容 OpenAI 协议 |
| 沙箱 | AST + 受限 exec | 内置 | 无需额外依赖，足够本地桌面场景 |
| 数据库 | SQLite + sqlite3 | 内置 | 零配置关系存储 |
| 向量存储 | ChromaDB | 0.4+ | 嵌入式模式，用于 Schema 相似度匹配 |
| 打包 | PyInstaller | 6.0+ | 单文件/单目录分发 |

## 开发环境搭建

### 前置条件
- Python 3.10 或更高（必须，3.9 不兼容部分类型注解）
- Windows 10/11（目标平台）
- Git（可选但推荐）

### 初始化步骤

```bash
# 1. 创建项目目录
mkdir MorphSheet
cd MorphSheet

# 2. 创建虚拟环境
python -m venv venv
venv\Scripts\activate

# 3. 安装依赖（见下方 requirements.txt）
pip install -r requirements.txt

# 4. 创建 .env 文件
echo DEEPSEEK_API_KEY=sk-your-key-here > .env
echo DEEPSEEK_BASE_URL=https://api.deepseek.com/v1 >> .env
echo DEEPSEEK_MODEL=deepseek-chat >> .env

# 5. 验证启动
python run.py
```

## 命名规范

- **Python 文件/模块**：`snake_case`（如 `file_reader.py`, `llm_client.py`）
- **Python 类**：`PascalCase`（如 `AgentOrchestrator`, `SkillStore`）
- **Python 函数/变量**：`snake_case`（如 `read_file()`, `source_schema`）
- **JS 文件**：`kebab-case`（如 `diff-view.js`，因为 URL 习惯）
- **JS 函数**：`camelCase`（如 `uploadFile()`, `renderDiff()`）
- **CSS 类名**：`kebab-case`（如 `.diff-view`, `.status-panel`）
- **API 路径**：`kebab-case`（如 `/api/upload-file`, `/api/skill-library`）
- **数据库表名**：`snake_case` 复数（如 `conversion_tasks`, `skill_templates`）

## config.py 设计

```python
# app/config.py
from pydantic_settings import BaseSettings

class Settings(BaseSettings):
    deepseek_api_key: str
    deepseek_base_url: str = "https://api.deepseek.com/v1"
    deepseek_model: str = "deepseek-chat"
    max_retries: int = 3
    sandbox_timeout: int = 30          # 秒
    max_upload_size_mb: int = 50
    data_dir: str = "data"
    chroma_db_dir: str = "chroma_db"

    model_config = {"env_file": ".env", "env_file_encoding": "utf-8"}

settings = Settings()
```
